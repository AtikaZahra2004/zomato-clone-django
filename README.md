# 🍔 Food Ordering System (Zomato Clone)

## 📌 Description
This is a food ordering web application built using Django.  
It allows users to browse food items, view details, and simulate an online food ordering experience.

---

## 🚀 Features
- 🍽️ Food item listing
- 🔍 View food details
- 🖼️ Image display
- 📂 Category-based food items
- 🛒 Basic ordering functionality
- ⚙️ Admin panel for managing items

---

## 🛠️ Tech Stack
- Python
- Django
- HTML, CSS
- SQLite

---
